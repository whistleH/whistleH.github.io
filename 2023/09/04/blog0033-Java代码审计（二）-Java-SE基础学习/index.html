1.变量命名

很多博客里都不会提到的Java标识符命名规范：

  1. 可以使用数字、字母、下划线以及$作为变量命名的组成
  2. 不能以数字开头
  3. 区分大小写

补充这个点主要可以在Java命名$_POST这样的PHP形式变量，感觉非常有意思。
